Information about the data type of columns and parameters:
  • string represents a text with a maximum length of 256 characters.
  • string(XX) represents a text with a maximum length of XX characters.
  • int represents a strict integer value.
  • float represents a floating-point number (but can also return an integer).
  • date represents a strict date (without time).
  • time represents a strict time (without date).
  • datetime represents a full date with time.
  • url represents a complete URL.
  • enum(A,B,C) represents a fixed list of values (in this case, the field can only contain A, B, or C).
  • yesNoElement represents a boolean: yes/no values only.
There are also additional complex types, such as registerFlag or traineeSearchField. These complex types all have their own dedicated page, which is accessible by clicking on the data type label directly.
